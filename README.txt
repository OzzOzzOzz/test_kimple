Installez MariaDB
Executez ./install_db.sh (nécessite mot de mass pour sudo)
lancez le site avec la commande "php artisan serve"
Connectez vous à l'adresse indiqué par php artisan

Vous pouvez crée des posts, les editer et les suprimer.
Le site ne dispose pas d'un systeme d'autantification ni du jeu Snake.
